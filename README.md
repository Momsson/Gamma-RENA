# Gamma-RENA
Pokročilý model kinetiky relativistických ubiehajúcich elektrónov a spätnosäzbovej emisie gama žiarenia. Simuluje energetickú bilanciu v plynoch a pevných látkach integráciou procesov Bremsstrahlung a fotoabsorpcie. Nástroj na predikciu vysokoenergetických javov (TGF) a radiačnej záťaže v letectve, nadväzujúci na najnovší výskum Pasko et al. (2026)
